These are the steps to just get you to the most recent commit

  1. git checkout master
    • Make sure you're on the master branch. Please.
  2. git fetch -all
    • This will get ALL the latest files onto your system.
  3. git reset --hard origin/master
    • Don't forget the hard
  4. git pull origin master
    • This is it. This is the one.
